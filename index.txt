git是什么?
是最先进的分布式版本控制系统

版本控制系统分 集中式  和 分布式 ！
集中式代表：cvs 和 svn
分布式代表：git

集中式版本控制系统需要联网才能使用，中央处理器一旦崩溃、不能联网时，都不能工作，需要等待。
分布式版本控制系统不需要联网也能使用，只要在上传数据的时候，联网就可以。


git 命令

pwd                         显示当前目录
git init                    在当前目录下，创建git仓库
cd 文件名                   进入该文件目录
cd ..                       返回上一级目录
git add 文件名              把当前文件添加到git仓库中的缓存区域
git commit -m "信息提示"    把当前文件添加到git仓库，并生成新的版本号
